+ **Cooldown:** `Once Per Short Rest` -> `Once Per Combat`
+ **\[NEW\]:** No longer breaks *Invisibility* or *Sneaking*

## Brace (Heavy Crossbows, Longbows)

+ **Cooldown:** `Once Per Short Rest` -> `Once Per Combat`
+ **\[NEW\]:** No longer breaks *Invisibility* or *Sneaking*

## Prepare (Greataxes)

+ **Cooldown:** `Once Per Short Rest` -> `Once Per Combat`
+ **\[NEW\]:** No longer breaks *Invisibility* or *Sneaking*
